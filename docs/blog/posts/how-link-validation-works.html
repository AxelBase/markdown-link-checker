What Counts as a Valid Link?

Published: September 26, 2025 | Last Updated: September 26, 2025

Not all links are created equal. Our Markdown Link Checker uses clear rules to determine validity—so you can trust the results and fix what matters.

Link Types We Check

  • HTTP/HTTPS URLs
  • Relative paths
  • Anchor links (e.g., #section)
  • Reference-style links

Skipped or Ignored Links

Mailto links, localhost references, and custom schemes are skipped by default. You can override this with configuration settings.

FAQ

Are redirects considered valid?

Yes, but they’re flagged separately so you can decide whether to update them.

Can I ignore certain domains?

Yes. Use the ignore pattern field to exclude domains or paths.

What about anchor links?

We check anchor syntax but can’t verify target existence unless a virtual tree is defined.
